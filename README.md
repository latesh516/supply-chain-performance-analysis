# Supply Chain Performance Analysis

This project performs an end-to-end analysis of supply chain operations using Python, SQL, and Power BI.

## Project Objectives

The goal of this project is to analyze supply chain performance and identify operational improvement opportunities across:

- Product demand and revenue performance
- Inventory availability and stockout risks
- Supplier lead time and production contribution
- Manufacturing efficiency and defect rates
- Logistics costs and transportation performance

## Tools Used

- Python (Pandas)
- SQL
- Power BI
- Excel

## Project Workflow

1. Business Understanding and Requirement Analysis
2. Data Audit and Data Cleaning
3. Exploratory Data Analysis
4. Inventory and Demand Analysis
5. Supplier Performance Analysis
6. Manufacturing Efficiency Analysis
7. Logistics Cost Analysis
8. KPI Calculation
9. Power BI Dashboard Development

## Key Insights

- Skincare products generated the highest revenue (~$241K).
- Several high-demand SKUs showed stockout risk due to low inventory levels.
- Supplier lead time varies significantly across suppliers.
- Haircare products showed the highest defect rates.
- Air transportation had the highest logistics cost while sea transport was the most economical.

## Dashboard

The Power BI dashboard provides operational visibility across demand, inventory, supplier performance, manufacturing quality, and logistics cost metrics.
